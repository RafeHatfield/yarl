# Docs Archive (Historical)

Files in this folder are preserved for history and design archaeology. They may be outdated or superseded. Do not treat them as current truth; prefer the canonical docs in `docs/` and the root README/ROADMAP.

## Structure
- `phase-notes/` – Phase 13-22 implementation notes, analysis, and summaries
- `fix-notes/` – Bug fix summaries, architecture fixes, knockback/combat patches
- `releases/` – PR descriptions, release notes, distribution audits, cleanup checklists
- `development/` – Stale session handoff files, debug mode proposals, refactor plans
- `bug-fixes/` – Earlier bug fix documentation
- `completed-features/` – Feature completion summaries
- Legacy design docs (boss fights, monster loot, power system, slime system, turn manager)
